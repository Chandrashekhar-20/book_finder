# Book Finder App

A simple React app to search books using the OpenLibrary API.

## Features
- Search by title
- Displays cover, author, and publication year
- Responsive plain CSS styling

## Commands
```
npm install
npm run dev
```

## API Used
OpenLibrary Search API: https://openlibrary.org/search.json?title={bookTitle}

## Deployment
Can be hosted easily on Vercel, Netlify, or CodeSandbox.
